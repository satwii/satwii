### 🚀 Featured Projects

**[Sawaari — Hyperlocal AI Ride-Sharing Platform](https://github.com/satwii/Shinchan-s-Sawaari)**
Full-stack carpooling/cab-splitting platform with a multilingual voice AI assistant (Azure Speech + Groq LLaMA-3.3-70B), OpenStreetMap route-overlap matching, real-time GPS tracking, Socket.io group chat, women-only Pink Mode, and Aadhaar verification.

**[PRISM — Real-Time Psychological Scam Detection](https://github.com/satwii/Prism)**
NLP system detecting manipulative intent (authority, urgency, grooming) in chat messages via a BERT-teacher → DistilBERT-student distillation pipeline (96.39% accuracy), with ChromaDB RAG retrieval for real-time alerting.

**[CrowdPulse — Real-Time Crowd Analysis & Stampede Prediction](https://github.com/satwii/CrowdCount)**
Hybrid crowd-risk system combining CSRNet density estimation with classical CV, on a Kafka + Spark + HDFS big-data pipeline processing 3000+ frames, using K-Means/GMM zoning and ARIMAX+LSTM forecasting for early danger alerts.

**Agrolite — Edge-Based Pollination Prediction** *(repo not yet public)*
Edge-AI system using ESP32 sensor data and a LightGBM-to-TensorFlow-Lite distilled model for offline, on-device agricultural decision support (>98% edge accuracy, ~0.03ms latency). Patent filed.
